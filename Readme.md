# EventSyncPro

A Real-Time Event Management System built with Flutter and Firebase.

## Features

- Real-time event updates via Firestore
- Role-based permissions (Admin, User)
- Offline support (Hive)
- Media upload (images, videos) with compression
- Email/password authentication (Firebase)
- Advanced UI: Carousel, countdown timer, video player

## Getting Started

1. Add your Firebase configuration files to `android/app` and `ios/Runner`.
2. Run `flutter pub get`.
3. Use `flutter run` to start the app.

See `EventSyncPro_Assignment.md` for detailed project description.
